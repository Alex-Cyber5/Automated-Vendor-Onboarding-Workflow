# Automated-Vendor-Onboarding-Workflow
Working on an AI Agentic workflow to streamline and automate the vendor onboarding process by capturing vendor information, assessing inherent risk, initiating the appropriate due diligence activities, assigning tasks, notifying stakeholders, and tracking progress through to completion.

# Project Definition
The Automated Vendor Onboarding Workflow is an end-to-end automation solution designed to streamline the (third-party) vendor onboarding process. The workflow captures new vendor intake requests, evaluates inherent risk using predefined security criteria, routes vendors through risk-based approval paths, distributes Due Diligence Questionnaires (DDQs), creates onboarding tasks, sends stakeholder notifications, and tracks progress from submission through completion. The objective is to reduce  manual effort, improve consistency, accelerate onboarding, and ensure that security and compliance requirements are embedded into every stage of the vendor onboarding lifecycle.
